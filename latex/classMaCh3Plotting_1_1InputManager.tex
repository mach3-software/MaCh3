\doxysubsection{Detailed Description}
This guy talks to the input files and is intended to smooth over the fact that our OA fitters use such a beautiful and diverse range of output file structures. 

Intended use for this class is to be called from inside of a \mbox{\hyperlink{classMaCh3Plotting_1_1PlottingManager}{Plotting\+Manager}} object which is what the user is intended to interact with, though in principle it should be strong and independent enough to be used all by itself. The interface to this object is intentionally pretty small as it is intended to deal with the complexities of the different input files pretty automatically. Usage should follow something like this\+:
