Factory function for creating SamplePDF and initialisation with systematic.

Template Parameters
SampleTypeThe class type of the sample to create, e.g., samplePDFTutorial.
Parameters
SampleConfigPath to sample config.
xsecA pointer to a covarianceXsec object for cross-section systematic settings.
osc(Optional) A pointer to a covarianceOsc object for oscillation systematic settings.
Returns
Vector of SampleType object, initialized and ready for use.
Note
Example
auto mySamples = MaCh3SamplePDFFactory<samplePDFTutorial>(SampleConfig, xsec, osc);
